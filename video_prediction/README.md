# EmotionAI - Live Video Prediction 🎥

Real-time facial emotion detection using your webcam.

This is the **live video prediction** version of the project, using a premium dark-themed design with glassmorphism.

## Technical Details
- **Architecture**: ResNet50V2
- **Backend API**: [Facial Emotion Detection API](https://huggingface.co/spaces/MuhammadSheraza002/facial-emotion-detection-backend)
- **Frontend**: Static HTML/CSS/JS (Modern Design)
